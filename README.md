# Avengers_battle_Simul
Interactive Avengers Battle Simulator built in C++ using OOPS principles. Features custom Iron Man suit mechanics, battle commands, upgrades, repairs, combat logs, power boosts, and real-time battle outcome evaluation.
# Avengers Battle Simulation

## Overview

Avengers Battle Simulation is a command-driven C++ project that models battles between Avengers and their enemies using custom-designed combat suits. The project demonstrates Object-Oriented Programming concepts such as encapsulation, operator overloading, class interactions, and STL container usage.

Each character is equipped with a suit possessing attributes such as Power, Durability, Energy, and Heat. During battle, characters can attack opponents, repair suits, upgrade equipment, boost power levels, and monitor battle status through an event-driven command system.

## Suit Attributes

Each suit maintains:

* **Power (P)**
* **Durability (D)**
* **Energy (E)**
* **Heat (H)**

Suits can become:

* **Overheated** when heat exceeds the threshold.
* **Destroyed** when durability reaches zero.
